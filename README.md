# Open-SSPM Website

Static website for [Open-SSPM](https://github.com/open-sspm/open-sspm).

## Development

1. Install dependencies:
   ```bash
   npm install
   ```

2. Build CSS:
   ```bash
   npm run build
   ```

3. Watch CSS:
   ```bash
   npm run watch
   ```

## Deployment

This website is designed to be deployed to GitHub Pages. Simply push the contents to the `main` branch or configure GitHub Actions to deploy.
